## üîç Quick Reference

### Key Information

**API Key**: Get from [console.groq.com](https://console.groq.com)

**Model**: `llama-3.3-70b-versatile`

**Free Tier**: 14,400 requests/day

**Response Time**: 300-500ms

**Accuracy**: 95%

**Setup Time**: 5 minutes

### Key Commands

```bash
# Install dependencies
cd server && npm install

# Start server
cd server && npm run dev

# Test Groq
npm run test:groq

# Build Android
cd client && npm run build && npx cap sync android
```

### Key Files

```
server/src/services/groqParser.js  - Groq parser
server/src/index.js                - API endpoint
server/.env                        - API key (add yours)
test-groq-ocr.js                   - Test script
```
